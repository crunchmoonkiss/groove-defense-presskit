# Groove Defense Press Kit

Official press kit for Groove Defense.

## Asset Folders

| Folder | Purpose |
|--------|---------|
| `/assets/capsules` | Store capsule images (header images, banners, key art) |
| `/assets/screenshots` | In-game screenshots for press and media use |
| `/assets/logos` | Game logos, studio logos, and branding assets |
| `/assets/trailers` | Video trailers and gameplay footage |

## Usage

This site is hosted via GitHub Pages. The `index.html` file serves as the main press kit page.
